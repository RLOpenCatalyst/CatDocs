Installation
============
This is a documentation that gives the developer and the user an overview of how to install RLCatalyst on a local instance 

**Supported Platforms**

RLCatalyst is currently supported on

 * Ubuntu 16.04

 * Ubuntu 14.04

 * CentOS 7
